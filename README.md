# Machine-Learning-for-Breast-Cancer-Prediction
This repository contains Team 1’s machine learning project on breast cancer prediction using the Breast Cancer Wisconsin (Diagnostic) dataset. The goal is to build and compare supervised classification models that predict whether a tumor is benign or malignant, using reliable validation and clear interpretation of results.

For Milestone 2, we produced a fully executable Jupyter Notebook (.ipynb) that includes data acquisition, data wrangling and cleaning, exploratory data analysis (EDA), and visualizations. The notebook is organized and designed to run from top to bottom without errors in Google Colab. We also exported the final wrangled dataset as a cleaned CSV file that is ready for machine learning in Milestone 3. All project files, including the notebook, cleaned dataset, and this updated README, have been committed to our GitHub repository.
